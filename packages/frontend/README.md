# Igloo's frontend

## Requirements

- nodejs: see [.nvmrc](./.nvmrc)
- npm: >= 8.3.1

## Getting started

- Run `npm i` to install all the project dependencies
- Run `npm start` to start the project
- The project will open at http://localhost:3000/
